🇬b/EN: Synopsis

"Rose Among Locusts" is a vertical-scroll visual narrative. It follows Aurelia von Wald, an ambitious aristocrat of the Administratum, whose pampered life is shattered alongside her world under the sudden assault of a Tyranid Hive Fleet. It is a chronicle of vanity forged into the sheer will to survive in the grim darkness of the far future.

🛠 Technical Details

Format: Single-file HTML5/Tailwind CSS interactive web page.

Assets: AI-generated imagery (Stable Diffusion/DALL-E) specifically curated for the Warhammer 40k aesthetic.

Typography: Gothic-inspired serif fonts (Cinzel, EB Garamond).

🚀 How to Run Locally

Clone the repository or download the ZIP file.

Ensure all .png image files are in the same folder as index.html.

Open index.html in any modern web browser.
